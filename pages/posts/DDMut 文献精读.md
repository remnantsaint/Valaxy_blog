# DDMut: 使用深度学习预测突变对蛋白质稳定性的影响
  [DDMut: predicting effects of mutations on protein stability using deep learning](https://doi.org/10.1093/nar/gkad472)  
  
  没有源码
## 摘要
  理解突变对蛋白质稳定性的影响对于变异解读与优先级排序、蛋白质工程以及生物技术至关重要。尽管已付出大量努力，但对预测工具的社区评估仍揭示了持续存在的局限性，包括计算时间长、预测能力低，以及对导致不稳定突变的预测存在偏差。

  为填补这一空白，我们开发了DDMut，这是一种快速且准确的孪生网络，用于预测单点和多点突变引起的吉布斯自由能变化，通过利用正向突变和假设的反向突变来确保模型满足反向对称性。该深度学习模型整合了蛋白质局部三维环境的图结构表示，并结合卷积层和Transformer编码器构建而成。这种组合能够更好地捕捉原子间的距离模式，有效提取短程和长程相互作用。DDMut在非冗余的盲测集上表现优异，对单点突变的皮尔逊相关系数达到0.70（均方根误差RMSE：1.37 kcal/mol），对双/三重突变的相关系数也达到0.70（RMSE：1.84 kcal/mol），性能优于大多数现有方法。尤为重要的是，DDMut具有高度可扩展性，并且对导致稳定和不稳定的突变均表现出良好的反向对称性能。我们相信，DDMut将成为一个有用的平台，有助于更好地理解突变的功能后果，并指导合理的蛋白质工程设计。DDMut可免费通过网站和API获取，地址为：https://biosig.lab.uq.edu.au/ddmut。

## 引言
  蛋白质是自然界在进化过程中精心打造的多功能且动态的工具，用于协调生命活动所必需的一系列生化过程。它们参与了许多生物过程，包括细胞信号传导、增殖、代谢和细胞死亡。因此，了解蛋白质序列的改变如何影响其结构、功能和相互作用，并由此产生不同的表型，是至关重要的。

  表征突变的分子效应可以为理解其生物学结果提供关键见解。在过去几年中，由于数据的积累以及错义突变对蛋白质产生的细微影响，这类突变已被广泛研究。在蛋白质序列水平上的单个氨基酸改变，可能导致其三维结构发生局部的原子级变化，从而影响蛋白质折叠的动力学、稳定性、柔韧性及动态行为。尽管蛋白质结构建模工具已取得巨大进展，但目前的蛋白质结构预测工具仍难以准确捕捉这些变化。

  人们已投入大量精力来理解和预测蛋白质编码区突变的分子效应，然而大多数方法在通量上存在局限（难以实现全基因组范围和饱和突变分析），仅局限于预测单点错义突变的后果，并且由于数据本身存在的固有偏差，对稳定化突变的预测能力较差，而稳定化突变对于生物技术应用至关重要。

  DDMut工作流程。该方法包含四个步骤。首先，从不同来源收集整理数据集，并从RCSB PDB数据库中获取蛋白质结构。其次，生成并标准化一组能够捕捉几何和理化特性的特征。然后，将这些特征输入神经网络，并根据训练表现对超参数和网络层数进行调优，再在非冗余的盲测数据集上进行验证。最后，将预测模型作为易于使用的网络界面免费提供。

## 材料和方法
### 数据集
#### 训练数据集
  用于预测单点突变效应的DDMut训练集，是从S2648和FireProtDB中整理而来。如果盲测集中的冗余条目（在突变水平上）与训练集具有相同的Uniprot ID和相同的突变，则会从训练集中将其移除。随后，对每个数据集内部的重复项（在突变水平上）进行去重处理，对于重复项，优先选择在最接近pH 7和25°C生理条件下测得的吉布斯自由能（ΔG）值。为了平衡ΔG的分布，按照以下方案向每个数据集中引入假设的反向突变，其中ΔG定义为蛋白质 unfolding（去折叠）的自由能变化
$$ΔΔG_{Mut->WT} = ΔG_{WT} - ΔG_{Mut} = -(ΔG_{Mut} - ΔG_{WT}) = -ΔΔG_{WT->Mut}$$
  
  这最终形成了我们的DDMut单点突变训练集S9028（包含153种蛋白质上的9028个突变）。S9028的ΔG分布如补充图S1所示。为了预测多点突变的效应，我们从DynaMut2的训练集中移除了多突变水平上的重复和冗余条目，并加入了反向突变，从而得到训练集SM1242（包含94种蛋白质的98个结构）。我们还进一步采用了基于蛋白质水平的非冗余划分方法，最终得到训练集SM1218（包含60种蛋白质的67个结构）。
#### 盲测数据集
  对于单点突变，我们采用了五个广泛用于现有蛋白质稳定性预测工具的通用非冗余盲测集，以进行基准性能比较。S276和S669包含与原始ProTherm数据集序列相似性较低的蛋白质，且在蛋白质水平上非冗余；而S1342是在突变水平上非冗余的盲测集，这意味着该数据集中的突变可能出现在训练集中已包含的同一种蛋白质上，但位于不同位置，或在同一位置但突变为不同的氨基酸残基。此外，我们还评估了来自CAGI5挑战赛（包括PTEN和TPMT的变异体）以及Gerasimavicius等人的深度突变扫描（DMS）数据集，后者包含了在45个独立实验中对161,441个变异体的功能评分。在移除同一测试集内部的冗余数据（在突变水平上）并加入假设的反向突变后，我们最终的盲测集分别包含552个突变（37种蛋白质上的37个结构）、1,304个突变（87种蛋白质上的94个结构）和2,024个突变（120种蛋白质上的129个结构）。这三个数据集在突变水平和蛋白质水平上的重叠情况如补充图S2所示。由于CAGI5挑战赛的数据是通过与突变蛋白融合的EGFP丰度来推断蛋白质稳定性变化的，因此对于PTEN（3,736个突变）和TPMT（3,627个突变）数据集，未包含假设的反向突变。同样，对于Gerasimavicius等人提供的DMS数据集，也未包含反向突变，因为这些数据反映的是突变的功能影响，而非蛋白质稳定性的热力学循环本身。
  
  用于预测多点突变的盲测数据集最初是在DynaMut2中报道的。通过移除重复项并加入反向突变，我们构建了多点突变盲测集SM420（包含63种蛋白质中的61个结构，共420个双突变和三突变）。在应用蛋白质水平的非冗余划分后，盲测集SM444包含了44种蛋白质中的44个结构。
  
  所有数据集的野生型结构均从蛋白质数据库（Protein Data Bank，PDB）下载，突变体结构则使用MODELLER软件及其默认的最小化流程，由相应的野生型结构生成。野生型和突变体结构均被用于生成正向和反向突变的特征。
### 特征工程
  生成了两组特征：基于图的特征（graph-based signatures）和互补特征（complementary features）：
  
  基于图的特征是使用mCSM生成的，该方法采用一种在残基局部环境的图结构表示内运行的“截断扫描算法”（Cut-off Scanning Algorithm），用于捕捉成对原子之间的距离模式。这些原子被标记为八种不同的药效团类型：疏水基团（Hydrophobics）、正电基团（Positives）、负电基团（Negatives）、氢键受体（Hydrogen Acceptors）、氢键供体（Hydrogen Donors）、芳香基团（Aromatics）、硫原子（Sulphurs）和中性基团（Neutrals）。
  
  互补特征既包括基于序列的特征也包括基于结构的特征：(a) 基于序列的特征是使用替换矩阵计算得出的，比如AAindex，用于捕捉理化和生化性质的变化；还有基于序列比对的BLOSUM和PAM矩阵。(b) 基于结构的特征则包含了溶剂可及性、残基深度、二级结构，以及通过Arpeggio工具计算的目标残基与邻近残基之间的原子间相互作用。此外，还包括突变后这些相互作用发生的变化。每组互补特征所使用的计算工具在补充表S1中有详细说明。这表示研究中综合运用了多种生物信息学方法和工具来全面评估蛋白质突变的影响。 
  
  生成的特征随后通过其均值和标准差进行标准化处理。
### 网络架构
  对于单点突变和多点突变这两种预测任务，DDMut模型均使用孪生网络（Siamese Networks）进行训练（见补充图S3），其中两个具有相同结构和共享权重的子网络分别处理由正向突变和反向突变计算得到的特征。	
  
  在每个子网络中：基于图的特征（graph-based signatures）首先通过卷积层进行处理，然后输入到一个Transformer编码器中；互补特征（complementary features）则通过两个全连接（密集）层进行处理。随后，这两部分处理后的特征被拼接（concatenated）在一起，并结合残差连接（residual connections），再经过一个全连接层进行整合。
  
  最后，采用一种改编并改进自文献(An antisymmetric neural network to predict free energy changes in protein variants)的对比损失函数（contrastive loss function）。该损失函数不仅考虑了预测值与实际ΔG值之间的误差，还同时考虑了反向对称性约束，即正向突变与其对应反向突变的预测结果之间的误差（要求ΔG_forward ≈ -ΔG_reverse），从而确保模型满足热力学上的反向对称性原则。
$$\begin{aligned}
\text { Loss }= & \log \cosh \left(\frac{\Delta \Delta G_{\text {Forward }}-\Delta \Delta G_{\text {Reverse }}}{2}-y\right) \\
& +\left|\Delta \Delta G_{\text {Forward }}+\Delta \Delta G_{\text {Reverse }}\right|,
\end{aligned}$$
  其中，$ΔG_{Forward}$ 和 $ΔG_{Reverse}$ 分别表示模型对正向突变及其对应反向突变的预测值，y 是正向突变的实验测得的 ΔG 值。对于一个完全满足反向对称性且预测准确的模型，以下规则将被满足：$ΔΔG_{Forward} = y, ΔΔG_{Reverse} = -ΔΔG_{Forward}$
  这将导致损失值为0。
  
  在训练阶段，根据模型在训练集上的交叉验证表现，对网络架构的超参数进行了精细调整。该过程针对单点突变模型和多点突变模型分别独立进行。所采用的评估指标包括：
  * 皮尔逊相关系数（Pearson’s correlation, r）
  * 肯德尔相关系数（Kendall’s correlation, τ 或 k）
  * 斯皮尔曼相关系数（Spearman’s correlation, ρ 或 s）
  * 均方根误差（Root Mean Square Error, RMSE）
  * 平均绝对误差（Mean Absolute Error, MAE）
  * 平均符号误差（Mean Signed Error, MSE）

  当模型的预测完全准确且满足反向对称性时，上述损失函数的值为零，表示达到了理想的预测状态。
## 验证
### 预测单点突变
  我们在包含9,028个单点突变的训练集上，通过10折交叉验证评估了DDMut的性能。采用了两种不同的数据划分策略：基于氨基酸水平的低冗余划分和基于蛋白质水平的低冗余划分。在氨基酸水平低冗余划分下，我们的方法达到了0.77的皮尔逊相关系数（RMSE：1.25 kcal/mol）；在蛋白质水平低冗余划分下，相关系数为0.70（RMSE：1.37 kcal/mol）（图2A）。两种划分策略下性能相近，表明模型具有良好的鲁棒性。DDMut在正向突变和反向突变上的预测表现也高度一致（RMSE分别为1.36和1.38 kcal/mol），且正向与对应反向突变的预测值之间的皮尔逊相关系数为-0.93，表明模型具有很高的反向对称性（补充图S11）。为了构建一个能够更稳健地预测更广泛蛋白质突变稳定性的模型，神经网络的超参数是基于蛋白质水平低冗余划分下的交叉验证表现进行调优的。最终的模型在完整的训练集上进行训练，然后在盲测集上进行评估。
  
  为了公平地将我们的模型与其他现有方法进行比较，我们在非冗余的盲测集上测试了DDMut，这些盲测集分别包含276、1,304和2,024个突变，以及来自CAGI5挑战赛的DMS数据集（包括PTEN和TPMT的变异体）。在四个盲测集中，我们的模型在其中三个上取得了最佳性能（图2B），并且在正向突变与假设的反向突变上，以及在稳定化突变（ΔG ≥ 0 kcal/mol）和不稳定化突变（ΔG < 0 kcal/mol）上均表现出一致的性能（图2B，补充表S3–S9）。这增强了我们对DDMut模型泛化能力的信心。随后，我们进一步评估了DDMut在预测功能得分方面的能力，共涉及45项独立的深度突变扫描（DMS）实验中的161,441个变异体，这些实验类型涵盖蛋白质丰度、蛋白质结合、活性测定、生长实验和病毒复制等（20）。与另外九种蛋白质稳定性预测工具相比，DDMut表现出具有竞争力的性能，但其在不同实验类型间的性能差异较大（补充图S12）。
  
  我们通过消融实验（ablation studies）评估了网络架构中每个组成部分的贡献。通过在模型中逐一禁用各个子模块，我们发现所有组件对于最终的预测都是必不可少的。其中，处理互补特征的两个全连接层对最终性能的贡献相对更大，而Transformer编码器和卷积层的贡献相对较小（补充表S10）。这种差异也可能与它们各自处理的特征类型有关。为了进一步理解是什么因素决定了一个突变是稳定化（ΔG ≥ 0 kcal/mol）还是不稳定化（ΔG < 0 kcal/mol），我们采用了一种与模型无关的方法——特征排列重要性（feature permutation importance），来评估每个特征对神经网络的重要性。具体做法是：在平衡的盲测集S1304上，我们随机打乱某一个特征在所有突变样本中的取值（同时保持该特征的均值和方差不变），然后观察模型性能（如皮尔逊相关系数）的下降程度。值得注意的是，排名前两位的最重要特征——“疏水相互作用”（hydrophobic contacts）和“疏水原子”（hydrophobic atoms）——都与突变引起的疏水性变化密切相关。单独打乱其中任意一个特征，都会导致模型在正向突变上的皮尔逊相关系数下降约0.05（补充表S11），这凸显了疏水作用在决定蛋白质突变稳定性效应中的核心作用。
